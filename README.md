# keymaker
demo：https://keymaker.iskwen.com/

## 簡介
kemaker是一個穩定、快速、支持高並發、免費的API接口平台

基於SpringCloud的分佈式微服務系統，通過Rebbion和OpenFeign實現負載均衡，所有微服務通過註冊進入consul實現集群管理，redis緩存突破database並發瓶頸，最後hystrix實現控流、降級和熔斷進一步保障系統高可用、高並發。
## 特色
* 分佈式微服務
* 集群部署
* 負載均衡
* 流量控制、降低、熔斷
* Redis二級緩存

## 運用
* SpringCloud
* Mybatis
* Druid
* Consul
* Hystrix
* OpenFeign
* Rebbion
